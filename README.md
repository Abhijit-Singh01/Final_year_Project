# Smart Cart - eCommerce Project

[![Django](https://img.shields.io/badge/Django-3.2-green)](https://www.djangoproject.com/)
[![Python](https://img.shields.io/badge/Python-3.14-blue)](https://www.python.org/)
[![Bootstrap](https://img.shields.io/badge/Bootstrap-5.3-purple)](https://getbootstrap.com/)
[![Razorpay](https://img.shields.io/badge/Razorpay-Payment-orange)](https://razorpay.com/)

Smart Cart is a modern **eCommerce web application** built with Django. Users can browse products, add items to the cart, and make secure payments online with Razorpay. The app is fully responsive and easy to use.

## Features

- User registration and login
- Browse products by category
- Add or remove items from the cart
- Checkout and payment integration with **Razorpay**
- Admin panel to manage products, categories, and orders
- Responsive design for desktop and mobile devices

---

## Tech Stack

- **Backend:** Django  
- **Frontend:** HTML, CSS, JavaScript, Bootstrap  
- **Database:** SQLite (Django default)  
- **Payment Gateway:** Razorpay
